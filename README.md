# 🌍 WanderLust

WanderLust is a full-stack travel accommodation web application inspired by Airbnb. Users can explore unique stays, create and manage listings, upload images, and share their experiences through reviews. The application is built using the MVC architecture and follows secure authentication and authorization practices.

## ✨ Features

- 🔐 User Authentication (Sign Up, Login & Logout)
- 🏡 Browse all travel listings
- ➕ Create new listings
- ✏️ Edit and update your own listings
- 🗑️ Delete your own listings
- ⭐ Add and delete reviews
- 📸 Upload listing images with Cloudinary
- 🔒 Authentication & Authorization using Passport.js
- ⚠️ Server-side validation using Joi
- 🍪 Session management using Express Session and MongoDB
- 📱 Responsive user interface

- Users can add reviews to listings.
- Users can delete only their own reviews.
